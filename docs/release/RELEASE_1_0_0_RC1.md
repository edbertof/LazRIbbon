# LazRibbon 1.0.0-rc1

This is the first release candidate for LazRibbon 1.0.

## Base

Promoted from the validated LazRibbon 0.3.67 line.

## Target environment

- Lazarus 4.6
- LCL
- Windows as the primary validation platform

## Scope freeze

No new large features should be added before 1.0 final. Accepted changes are limited to:

- compilation blockers;
- installation or design-time registration blockers;
- confirmed regressions in demos;
- confirmed regressions in BackStage, SkinManager, SkinGallery, Quick Access Toolbar, or TLazRibbonForm;
- essential documentation or packaging fixes.

## Not in scope before 1.0 final

- deep refactoring of LazRibbon_Core.pas;
- deep refactoring of LazRibbon_Backstage.pas;
- replacing the custom XML parser;
- global compiler mode conversion;
- feature parity with commercial Ribbon suites.
